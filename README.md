# README


<h2>Utilisation en local</h2>

```shell
git clone https://github.com/NicolasHov/Eventbrite-like-of-the-world.git
```
```shell
cd Eventbrite-like-of-the-world
```
```shell
bundle install 
```
```shell
rails db:create
```
```shell
rails db:migrate
```
```shell
rails db:seed
```
```shell
rails server  
```

Go sur http://localhost:3000/
